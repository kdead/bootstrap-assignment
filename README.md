# Bootstrap Assignment

## About

A responsive web page built with Bootstrap 5 demonstrating forms, tables, images, and navigation components.

## How to View

1. Clone or download this repository
2. Open `index.html` in any web browser
3. No installation or setup required

## Features

- Responsive navigation bar with hamburger menu
- Registration form with grid layout
- Data table with striped and hoverable rows
- Responsive images including a circular profile photo
- Responsive button utilities
